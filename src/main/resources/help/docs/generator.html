Password Generator



  1. The length of the password.
  2. If you select "readable"  passwords, only characters a-z will be used. This is less safe, but it is easier to remember the passwords. It depends which level of security you want.
  3. Upper/lower case will be used if selected.
  4. Digits will be used if selected.
  5. Punctuation will be used if selected. Not all applications support such passwords.
  6. The category gives an indication of the safety level of the generated passwords. This is only indicative.
The more "unreadable" you make the passwords the safer they are, but the trade-off is that they are difficult to remember.
