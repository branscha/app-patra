Password Editor

There are two different edit screens, one for version 1.0 entries and one for 2.0 entries. They only differ slightly, the group field is the only difference.




  1. The title of the entry. This is the most important discerning characteristic of an entry.
  2. The uid associated with this entry.
  3. The password.
  4. You can generate a password using the default settings (see the preferences).
  5. You can also generate a password while overriding the default settings temporarily. It might be necessary because some application requests specific password formats which are not supported by the default settings. Some applications do not support punctuation for example.
  6. This option lets you make the password visible (readable).
  7. Some notes you can add to the entry, for your own use.
